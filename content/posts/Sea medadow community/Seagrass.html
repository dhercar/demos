Introduction

In this document, the direct and indirect interactions in a hypothetical community are studied form a theoretical perspective. A simulation has been created to observe the system behaviour. To put some context, the interacting species are some of the commonly found in a Mediterranean seagrass meadow dominated by Posidonia oceanica.

The most common interaction type has been set to be competence for resources. Two clusters of organisms are observed for this interaction: The primary producers and all filtering organisms. The former cluster integrates Cymodocea nodosa (c_nod), Halophyla stipulacea (h_sti) and Posidonia oceanica (p_oce) , but red algae (alga) could also compete for resources such as light when the seagrass growth is too high. However, as the seagrass stabilizes the sandbed, allowing benthic organisms like red algae to proliferate, the relationship of red algae with all three seagrass species is taken to be positive. In addition, epibiontic (epib) organisms may also interact with P. oceanica by blocking the light and nutrient exchange and enhancing predatory pressure. Thus, the interaction is set to be negative. The latter cluster created by competence interactions involves all 5 filtering organisms ( epibiontic organisms, sponges (pori), Pinna nobilis (p_nob), bivalve (biva) and sabellida polychaeta (espi)).

All three seagrass species have been considered to provide a suitable habitat not only for the red algae but also for P. nobilis, porifera and sabellida. This has been represented as a positive facilitation interaction. In the case of epibiontic organisms, only P. oceanica has been set to provide this kind of positive interaction due to the longevity of their leafs and its slower growth compared to C. nodosa and H. stipulacea.

Finally, a predator - prey interaction takes place between the sea urchin Paracentrotus lividus (p_liv) and all plant organisms ( the three seagrass species and the red algae).

Comunity interaction model

A general expression to describe interactions has been set as a reference to develop a multi-species interaction model.

\[\frac{\text{d}C_{1}}{\text{d}t} = r_1C_{1}\left( 1- \frac{C_{1} - \alpha_{21} C_{2}}{k_1} \right)\] \[\frac{\text{d}C_{2}}{\text{d}t} = r_2C_{2}\left( 1- \frac{C_{2} - \alpha_{12} C_{2}}{k_2} \right)\]

where:

  • \(C_{i}\) represents the abundance of the species C.
  • \(t\) represents the time.
  • \(r_{i}\) represents the intrinsic growth rate.
  • \(\alpha_{ij}\) Modules the weight of the interaction that \(i\) exerts on \(j\).
  • \(K_i\) represents the carrying capacity.

As the system we aim to study has 10 species, a species abundance vector (instead of 10 different individual values) is used as state variable. Consequently, all parameters but \(\alpha\) are also included as vectors. \(\alpha\) values are represented in the interaction matrix already created.

Thus, the current model is built as follows:

\[\frac{\text{d}\vec{C_{i}}}{\text{d}t} = \vec{r_i}\vec{C_{i}}\left( 1- \frac{\vec{C_{i}} - [\alpha_{ji}] \vec{C_{j}}}{\vec{k_i} }\right)\]


library(deSolve)

r <- rep(0.2, nSp)
k <- rep(1, 10)

state<- rep(0.4, 10) 
times<-seq(0,50,0.5)

par <- c(c_nod, h_sti, p_oce, epib, pori, p_nob, biba, espir, p_liv, alga, r, k) 
 MODEL_1 <- function(t, state, parameters) {  
  with(as.list(c(state, parameters)), {
   
  C = matrix(state[1:nSp], nrow = nSp, ncol=1) 
  r= matrix(c(parameters[(nSp*nSp+1):(nSp*nSp + nSp)]), nrow= nSp, ncol=1)
  I= t(matrix(c(parameters[1:(nSp*nSp)]), nrow= nSp)) #Interaction matrix
  k= matrix(c(parameters[(nSp*nSp+nSp +1):(nSp*nSp + 2*nSp)]), nrow= nSp, ncol=1)

  dC = r*C*( 1 - (C - 0.2*I%*%C)/k)
  
      return(list(c(dC)))
  })
 }

  out <- ode(state, times, MODEL_1, par)



Model result

Equal values are given to each species parameters (\(r\) = 0.2 and \(k\) = 1). Initial state is also even among species (\(C_{t=0}\) = 0.4). Fig. 2 shows the model simulation for a period of 50 units of time. At the end of this period all species abundances seem to stabilize with great differences in their value.
